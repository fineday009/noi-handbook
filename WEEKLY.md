# 每周刷题目标与完成记录

## 使用说明

- **每周一（手动）**：在文件末尾新增本周小节，照抄模板格式，填入"目标"题号。
  - 支持范围写法 `1111-1141`，也支持逗号分隔 `1111, 1113, 1120`，可混用。
- **每周日 20:00（自动）**：定时任务会统计目标中哪些题的 `<题号>.cpp` 已提交到仓库，
  自动填写该小节的"完成情况"并推送。**统计后记得 `git pull` 拉取结果。**

新一周的小节模板（复制后改日期和目标即可）：

```
## Week of YYYY-MM-DD

- 目标：
- 完成情况：（待自动统计）
```
